## What is CommuCore?

CommuCore is a modern open-source management system for associations, clubs, NGOs and community organizations.

It helps you organize everything in one place:

- Member management
- Accounts & bookings
- Event planning and ticket registrations
- Public reports and blog entries
- Email communication
- Multilingual frontend & backend
- Built with Laravel, Livewire 3 and FluxUI.
