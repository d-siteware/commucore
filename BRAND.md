# CommuCore Brand Guidelines

---

## Brand Name

CommuCore

Always written as:

CommuCore

Not:

commucore  
Commu Core  

---

## Brand Meaning

CommuCore represents the technical and organizational core of communities and associations.

Values:

- Structure
- Transparency
- Community
- Stability
- Openness

---

## Logo

The logo represents:

- a central core
- connected modular segments
- organizational structure

---

## Logo Usage

Allowed:

- GitHub
- Documentation
- Application UI
- Website

Do not:

- distort
- recolor
- rotate
- apply effects
- change proportions

---

## Color Palette

Primary:

- Core Blue: #2563EB
- Community Teal: #0D9488

Neutral:

- Background: #F8FAFC
- Surface: #FFFFFF
- Text: #0F172A
- Muted: #64748B
- Border: #E2E8F0

---

## Typography

Recommended fonts:

- Inter
- Space Grotesk
- IBM Plex Sans

Avoid decorative fonts.

---

## Tone & Voice

- Clear
- Calm
- Professional
- Respectful
- Inclusive

Avoid marketing exaggeration.

---

## Usage Rights

The CommuCore name and logo may be used for:

- project-related content
- documentation
- contributions

Commercial use outside the software context requires permission.
